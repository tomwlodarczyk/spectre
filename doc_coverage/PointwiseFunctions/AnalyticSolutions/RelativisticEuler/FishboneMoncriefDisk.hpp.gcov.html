      34             :  * \brief Fluid disk orbiting a Kerr black hole
      35             :  *
      36             :  * The Fishbone-Moncrief solution to the 3D relativistic Euler system
      37             :  * \cite fishbone1976apj, representing the isentropic flow of a thick fluid disk
      38             :  * orbiting a Kerr black hole. In Boyer-Lindquist coordinates
      39             :  * \f$(t, r, \theta, \phi)\f$, the flow is assumed to be purely toroidal,
      40             :  *
      41             :  * \f{align*}
      42             :  * u^\mu = (u^t, 0, 0, u^\phi),
      43             :  * \f}
      44             :  *
      45             :  * where \f$u^\mu\f$ is the 4-velocity. Then, all the fluid quantities are
      46             :  * assumed to share the same symmetries as those of the background spacetime,
      47             :  * namely they are stationary (independent of \f$t\f$), and axially symmetric
      48             :  * (independent of \f$\phi\f$). Note that \f$r\f$ and \f$\theta\f$ can also be
      49             :  * interpreted as Kerr (a.k.a. "spherical" Kerr-Schild) coordinates
      50             :  * (see gr::KerrSchildCoords), and that the symmetries of the equilibrium ensure
      51             :  * that \f$u^\mu\f$ as defined above is also the 4-velocity in Kerr coordinates.
      52             :  *
      53             :  * Self-gravity is neglected, so that the fluid
      54             :  * variables are determined as functions of the metric. Following the treatment
      55             :  * by Kozlowski et al. \cite Kozlowski1978aa (but using signature +2)
      56             :  * the solution is expressed in terms of the quantities
      57             :  *
      58             :  * \f{align*}
      59             :  * \Omega &= \dfrac{u^\phi}{u^t},\\
      60             :  * W &= W_\text{in} - \int_{p_\text{in}}^p\frac{dp}{e + p},
      61             :  * \f}
      62             :  *
      63             :  * where \f$\Omega\f$ is the angular velocity, \f$p\f$ is the fluid pressure,
      64             :  * \f$e\f$ is the energy density, and \f$W\f$ is an auxiliary quantity
      65             :  * interpreted in the Newtonian limit as the total (gravitational + centrifugal)
      66             :  * potential. \f$W_\text{in}\f$ and \f$p_\text{in}\f$ are the potential and
      67             :  * the pressure at the radius of the inner edge, i.e. the closest edge to the
      68             :  * black hole. Here we assume \f$p_\text{in} = 0.\f$ The solution to the Euler
      69             :  * equation is then
      70             :  *
      71             :  * \f{align*}
      72             :  * (u^t)^2 &= \frac{A}{2\Delta\Sigma}\left(1 +
      73             :  * \sqrt{1 + \frac{4l^2\Delta \Sigma^2}{A^2\sin^2\theta}}\right)\\
      74             :  * \Omega &= \frac{\Sigma}{A (u^t)^2}\frac{l}{\sin^2\theta} + \frac{2Mra}{A}\\
      75             :  * u^\phi &= \Omega u^t\\
      76             :  * W &= l\Omega - \ln u^t,
      77             :  * \f}
      78             :  *
      79             :  * where
      80             :  *
      81             :  * \f{align*}
      82             :  * \Sigma = r^2 + a^2\cos^2\theta\qquad
      83             :  * \Delta = r^2 - 2Mr + a^2\qquad
      84             :  * A = (r^2 + a^2)^2 - \Delta a^2 \sin^2\theta
      85             :  * \f}
      86             :  *
      87             :  * and \f$l = u_\phi u^t\f$ is the so-called angular momentum per unit
      88             :  * intertial mass, which is a parameter defining an
      89             :  * individual disk. In deriving the solution, an integration constant has been
      90             :  * chosen so that \f$ W\longrightarrow 0\f$ as \f$r\longrightarrow \infty\f$,
      91             :  * in accordance with the Newtonian limit. Note that, from its definition,
      92             :  * equipotential contours coincide with isobaric contours. Physically, the
      93             :  * matter can fill each of the closed surfaces \f$W = \text{const}\f$, giving
      94             :  * rise to an orbiting thick disk. For \f$W > 0\f$, all equipotentials are open,
      95             :  * whereas for \f$W < 0\f$, some of them will be closed. Should a disk exist,
      96             :  * the pressure reaches a maximum value on the equator at a coordinate radius
      97             :  * \f$r_\text{max}\f$ that is related to the angular momentum per unit inertial
      98             :  * mass via
      99             :  *
     100             :  * \f{align*}
     101             :  * l = \dfrac{M^{1/2}(r_\text{max}^{3/2} + aM^{1/2})(a^2 - 2aM^{1/2}
     102             :  * r_\text{max}^{1/2} + r_\text{max}^2)}{2aM^{1/2}r_\text{max}^{3/2} +
     103             :  * (r_\text{max} - 3M)r_\text{max}^2}.
     104             :  * \f}
     105             :  *
     106             :  * Once \f$W\f$ is determined, an equation of state is required in order to
     107             :  * obtain the thermodynamic variables. If the flow is isentropic, the specific
     108             :  * enthalpy can readily be obtained from the first and second laws of
     109             :  * thermodynamics: one has
     110             :  *
     111             :  * \f{align*}
     112             :  * \frac{dp}{e + p} = \frac{dh}{h}
     113             :  * \f}
     114             :  *
     115             :  * so that
     116             :  *
     117             :  * \f{align*}
     118             :  * h = h_\text{in}\exp(W_\text{in} - W),
     119             :  * \f}
     120             :  *
     121             :  * and the pressure can be obtained from a thermodynamic relation of the form
     122             :  * \f$h = h(p)\f$. Here we assume a polytropic relation
     123             :  *
     124             :  * \f{align*}
     125             :  * p = K\rho^\gamma.
     126             :  * \f}
     127             :  *
     128             :  * Once all the variables are known in Boyer-Lindquist (or Kerr) coordinates, it
     129             :  * is straightforward to write them in Cartesian Kerr-Schild coordinates. The
     130             :  * coordinate transformation in gr::KerrSchildCoords helps read the Jacobian
     131             :  * matrix, which, applied to the azimuthal flow of the disk, gives
     132             :  *
     133             :  * \f{align*}
     134             :  * u_\text{KS}^\mu = u^t(1, -y\Omega, x\Omega, 0),
     135             :  * \f}
     136             :  *
     137             :  * where \f$u^t\f$ and \f$\Omega\f$ are now understood as functions of the
     138             :  * Kerr-Schild coordinates. Finally, the spatial velocity can be readily
     139             :  * obtained from its definition,
     140             :  *
     141             :  * \f{align*}
     142             :  * \alpha v^i = \frac{u^i}{u^t} + \beta^i,
     143             :  * \f}
     144             :  *
     145             :  * where \f$\alpha\f$ and \f$\beta^i\f$ are the lapse and the shift,
     146             :  * respectively.
     147             :  *
     148             :  * \note Kozlowski et al. \cite Kozlowski1978aa denote
     149             :  * \f$l_* = u_\phi u^t\f$ in order to
     150             :  * distinguish this quantity from their own definition \f$l = - u_\phi/u_t\f$.
     151             :  */
