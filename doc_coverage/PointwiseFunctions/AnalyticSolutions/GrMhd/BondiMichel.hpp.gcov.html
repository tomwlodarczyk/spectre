      27             :  * \brief Bondi-Michel accretion \cite Michel1972 with superposed magnetic field
      28             :  * in Schwarzschild spacetime in Cartesian Kerr-Schild coordinates.
      29             :  *
      30             :  * An analytic solution to the 3-D GRMHD system. The user specifies the sonic
      31             :  * radius \f$r_c\f$ and sonic rest mass density \f$\rho_c\f$, which are the
      32             :  * radius and rest mass density at the sonic point, the radius at which the
      33             :  * fluid's Eulerian velocity as seen by a distant observer overtakes the local
      34             :  * sound speed \f$c_{s,c}\f$. With a specified polytropic exponent \f$\gamma\f$,
      35             :  * these quantities can be related to the sound speed at infinity
      36             :  * \f$c_{s,\infty}\f$ using the following relations:
      37             :  *
      38             :  * \f{align*}
      39             :  * c_{s,c}^2 &= \frac{M}{2r_c - 3M} \\
      40             :  * c_{s,\infty}^2 &= \gamma - 1 + (c_{s,c}^2 - \gamma + 1)\sqrt{1 + 3c_{s,c}^2}
      41             :  * \f}
      42             :  *
      43             :  * In the case of the interstellar medium, the sound
      44             :  * speed is \f$\approx 10^{-4}\f$, which results in a sonic radius of
      45             :  * \f$\approx 10^8 M\f$ for \f$\gamma \neq 5/3\f$ \cite RezzollaBook.
      46             :  *
      47             :  * The density is found via root-finding, through the
      48             :  * Bernoulli equation. As one approaches the sonic radius, a second root makes
      49             :  * an appearance and one must take care to bracket the correct root. This is
      50             :  * done by using the upper bound
      51             :  * \f$\frac{\dot{M}}{4\pi}\sqrt{\frac{2}{Mr^3}}\f$.
      52             :  *
      53             :  * Additionally specified by the user are the polytropic exponent \f$\gamma\f$,
      54             :  * and the strength parameter of the magnetic field \f$B_0\f$.
      55             :  * In Cartesian Kerr-Schild coordinates \f$(x, y, z)\f$, where
      56             :  * \f$ r = \sqrt{x^2 + y^2 + z^2}\f$, the superposed magnetic field is
      57             :  * \cite Etienne2010ui
      58             :  *
      59             :  * \f{align*}
      60             :  * B^i(\vec{x},t) = \frac{B_0 M^2}{r^3 \sqrt{\gamma}}x^i
      61             :  *  =\frac{B_0 M^2}{r^3 \sqrt{1 + 2M/r}}x^i.
      62             :  * \f}
      63             :  *
      64             :  * The accretion rate is
      65             :  *
      66             :  * \f{align*}{
      67             :  * \dot{M}=4\pi r^2\rho u^r,
      68             :  * \f}
      69             :  *
      70             :  * and at the sonic radius
      71             :  *
      72             :  * \f{align*}{
      73             :  * \dot{M}_c=\sqrt{8}\pi \sqrt{M}r_c^{3/2}\rho_c.
      74             :  * \f}
      75             :  *
      76             :  * The polytropic constant is given by
      77             :  *
      78             :  * \f{align*}{
      79             :  * K=\frac{1}{\gamma\rho_c^{\gamma-1}}
      80             :  * \left[\frac{M(\gamma-1)}{(2r_c-3M)(\gamma-1)-M}\right].
      81             :  * \f}
      82             :  *
      83             :  * The density as a function of the sound speed is
      84             :  *
      85             :  * \f{align*}{
      86             :  * \rho^{\gamma-1}=\frac{(\gamma-1)c_s^2}{\gamma K(\gamma-1-c_s^2)}.
      87             :  * \f}
      88             :  *
      89             :  * #### Horizon quantities, \f$\gamma\ne5/3\f$
      90             :  * The density at the horizon is given by:
      91             :  *
      92             :  * \f{align*}{
      93             :  *  \rho_h\simeq\frac{1}{16}
      94             :  *  \left(\frac{5-3\gamma}{2}\right)^{(3\gamma-5)/[2(\gamma-1)]}
      95             :  *  \frac{\rho_\infty}{c_{s,\infty}^3}.
      96             :  * \f}
      97             :  *
      98             :  * Using the Lorentz invariance of \f$b^2\f$ we evaluate:
      99             :  *
     100             :  * \f{align*}{
     101             :  * b^2=\frac{B^2}{W^2}+(B^i v_i)^2=
     102             :  *  B^r B^r(1-\gamma_{rr}v^r v^r)+B^r B^r v^r v^r
     103             :  * =B^r B^r = \frac{B_0^2 M^4}{r^4},
     104             :  * \f}
     105             :  *
     106             :  * where \f$r\f$ is the Cartesian Kerr-Schild radius, which is equal to the
     107             :  * areal radius for a non-spinning black hole. At the horizon we get
     108             :  *
     109             :  * \f{align*}{
     110             :  * b^2_h=\frac{B^2_0}{16}.
     111             :  * \f}
     112             :  *
     113             :  * Finally, we get
     114             :  *
     115             :  * \f{align*}{
     116             :  * B_0 = 4 \sqrt{b^2_h} = 4\sqrt{\rho_h} \sqrt{\frac{b^2_h}{\rho_h}},
     117             :  * \f}
     118             :  *
     119             :  * where the last equality is useful for comparison to papers that give
     120             :  * \f$b^2_h/\rho_h\f$.
     121             :  *
     122             :  * To help with comparing to other codes the following script can be used to
     123             :  * compute \f$b^2_h/\rho_h\f$:
     124             :  *
     125             :  * \code{.py}
     126             :  * #!/bin/env python
     127             :  *
     128             :  * import numpy as np
     129             :  *
     130             :  * # Input parameters
     131             :  * B_0 = 18
     132             :  * r_c = 8
     133             :  * rho_c = 1 / 16
     134             :  * gamma = 4 / 3
     135             :  * mass = 1
     136             :  *
     137             :  * K = 1 / (gamma * rho_c**(gamma - 1)) * ((gamma - 1) * mass) / (
     138             :  *     (2 * r_c - 3 * mass) * (gamma - 1) - mass)
     139             :  * c_s_c = mass / (2 * r_c - 3 * mass)
     140             :  * c_inf = gamma - 1 + (c_s_c - gamma + 1) * np.sqrt(1. + 3. * c_s_c)
     141             :  * rho_inf = ((gamma - 1) * c_inf / (gamma * K *
     142             :  *                                   (gamma - 1 - c_inf)))**(1. / (gamma - 1.))
     143             :  * rho_h = 1. / 16. * (2.5 - 1.5 * gamma)**(
     144             :  *     (3 * gamma - 5) / (2 * (gamma - 1))) * rho_inf / (c_inf**1.5)
     145             :  *
     146             :  * print("B_0", B_0)
     147             :  * print("r_c: ", r_c)
     148             :  * print("rho_c", rho_c)
     149             :  * print("b_h^2/rho_h: ", B_0**2 / (16. * rho_h))
     150             :  * print("gamma: ", gamma)
     151             :  * \endcode
     152             :  *
     153             :  * #### Horizon quantities, \f$\gamma=5/3\f$
     154             :  * The density at the horizon is given by:
     155             :  *
     156             :  * \f{align*}{
     157             :  * \rho_h\simeq \frac{1}{16}\frac{\rho_\infty}{u_h c_{s,\infty}^3},
     158             :  * \f}
     159             :  *
     160             :  * which gives \cite RezzollaBook
     161             :  *
     162             :  * \f{align*}{
     163             :  * \rho_h\simeq 0.08\frac{\rho_\infty}{c_{s,\infty}^3}.
     164             :  * \f}
     165             :  *
     166             :  * The magnetic field \f$b^2\f$ is the same as the \f$\gamma\ne5/3\f$.
     167             :  */
