       6             : 
       7             : SpECTRE code is frequently run through the
       8             : [Clang Tidy](http://clang.llvm.org/extra/clang-tidy/) static analyzer.
       9             : Since analyzing a single source file can take over half a minute it
      10             : is generally not advisable to run clang-tidy over the entire code base.
      11             : If CMake isn't finding clang-tidy, make sure clang-tidy is installed and that
      12             : you have chosen clang as your compiler. To analyze a single source file
      13             : run, for example
      14             : `make clang-tidy FILE=/path/to/source/src/DataStructures/DataVector.cpp`.
      15             : To analyze the entire code base run `make clang-tidy-all`. To analyze all
      16             : changed C++ source files in the commits from `FIRST_HASH` to `HEAD`, run `make
      17             : clang-tidy-hash HASH=FIRST_HASH`.
