      23             : /*!
      24             :  *  \ingroup CoordinateMapsGroup
      25             :  *
      26             :  *  \brief Three dimensional map from the cube to a bulged cube.
      27             :  *  The cube is shaped such that the surface is compatible
      28             :  *  with the inner surface of Wedge3D.
      29             :  *  The shape of the object can be chosen to be cubical,
      30             :  *  if the sphericity is set to 0, or to a sphere, if
      31             :  *  the sphericity is set to 1. The sphericity can
      32             :  *  be set to any number between 0 and 1 for a bulged cube.
      33             :  *
      34             :  *  \details The volume map from the cube to a bulged cube is obtained by
      35             :  *  interpolating between six surface maps, twelve bounding curves, and
      36             :  *  eight corners. The surface map for the upper +z axis is obtained by
      37             :  *  interpolating between a cubical surface and a spherical surface. The
      38             :  *  two surfaces are chosen such that the latter circumscribes the former.
      39             :  *
      40             :  *  We make a choice here as to whether we wish to use the logical coordinates
      41             :  *  parameterizing these surface as they are, in which case we have the
      42             :  *  equidistant choice of coordinates, or whether to apply a tangent map to them
      43             :  *  which leads us to the equiangular choice of coordinates. In terms of the
      44             :  *  logical coordinates, the equiangular coordinates are:
      45             :  *
      46             :  *  \f[\textrm{equiangular xi} : \Xi(\xi) = \textrm{tan}(\xi\pi/4)\f]
      47             :  *
      48             :  *  \f[\textrm{equiangular eta}  : \mathrm{H}(\eta) = \textrm{tan}(\eta\pi/4)\f]
      49             :  *
      50             :  *  With derivatives:
      51             :  *
      52             :  *  \f[\Xi'(\xi) = \frac{\pi}{4}(1+\Xi^2)\f]
      53             :  *
      54             :  *  \f[\mathrm{H}'(\eta) = \frac{\pi}{4}(1+\mathrm{H}^2)\f]
      55             :  *
      56             :  *  The equidistant coordinates are:
      57             :  *
      58             :  *  \f[ \textrm{equidistant xi}  : \Xi = \xi\f]
      59             :  *
      60             :  *  \f[ \textrm{equidistant eta}  : \mathrm{H} = \eta\f]
      61             :  *
      62             :  *  with derivatives:
      63             :  *
      64             :  *  <center>\f$\Xi'(\xi) = 1\f$, and \f$\mathrm{H}'(\eta) = 1\f$</center>
      65             :  *
      66             :  *  We also define the variable \f$\rho\f$, given by:
      67             :  *
      68             :  *  \f[\rho = \sqrt{1+\Xi^2+\mathrm{H}^2}\f]
      69             :  *
      70             :  *  ### The Spherical Face Map
      71             :  *  The surface map for the spherical face of radius \f$R\f$ lying in the
      72             :  *  \f$+z\f$
      73             :  *  direction in either choice of coordinates is then given by:
      74             :  *
      75             :  *  \f[
      76             :  *  \vec{\sigma}_{spherical}(\xi,\eta) =
      77             :  *  \begin{bmatrix}
      78             :  *  x(\xi,\eta)\\
      79             :  *  y(\xi,\eta)\\
      80             :  *  z(\xi,\eta)\\
      81             :  *  \end{bmatrix}  = \frac{R}{\rho}
      82             :  *  \begin{bmatrix}
      83             :  *  \Xi\\
      84             :  *  \mathrm{H}\\
      85             :  *  1\\
      86             :  *  \end{bmatrix}
      87             :  *  \f]
      88             :  *
      89             :  *  ### The Cubical Face Map
      90             :  *  The surface map for the cubical face of side length \f$2L\f$ lying in the
      91             :  *  \f$+z\f$ direction is given by:
      92             :  *
      93             :  *  \f[
      94             :  *  \vec{\sigma}_{cubical}(\xi,\eta) =
      95             :  *  \begin{bmatrix}
      96             :  *  x(\xi,\eta)\\
      97             :  *  y(\xi,\eta)\\
      98             :  *  L\\
      99             :  *  \end{bmatrix}  = L
     100             :  *  \begin{bmatrix}
     101             :  *  \Xi\\
     102             :  *  \mathrm{H}\\
     103             :  *  1\\
     104             :  *  \end{bmatrix}
     105             :  *  \f]
     106             :  *
     107             :  *  ### The Bulged Face Map
     108             :  *  To construct the bulged map we interpolate between a cubical face map of
     109             :  *  side length \f$2L\f$ and a spherical face map of radius \f$R\f$, with the
     110             :  *  interpolation parameter being \f$s\f$, the `sphericity`.
     111             :  *  The surface map for the bulged face lying in the \f$+z\f$ direction is then
     112             :  *  given by:
     113             :  *
     114             :  *  \f[
     115             :  *  \vec{\sigma}_{+\zeta}(\xi,\eta) = \left\{(1-s)L + \frac{sR}{\rho}\right\}
     116             :  *  \begin{bmatrix}
     117             :  *  \Xi\\
     118             :  *  \mathrm{H}\\
     119             :  *  1\\
     120             :  *  \end{bmatrix}
     121             :  *  \f]
     122             :  *
     123             :  *  This equation defines the upper-z map \f$\vec{\sigma}_{+\zeta}\f$, and we
     124             :  *  similarly define the other five surface maps \f$\vec{\sigma}_{+\eta}\f$,
     125             :  *  \f$\vec{\sigma}_{+\xi}\f$, and so on by appropriate rotations.
     126             :  *  We constrain L by demanding that the spherical face circumscribe the cube.
     127             :  *  With this condition, we have \f$L = R/\sqrt3\f$.
     128             :  *
     129             :  *  ### The General Formula for 3D Isoparametric Maps
     130             :  *  The general formula is given by Eq. 1 in section 2.1 of Hesthaven's paper
     131             :  *  "A Stable Penalty Method For The Compressible Navier-Stokes Equations III.
     132             :  *  Multidimensional Domain Decomposition Schemes" available
     133             :  *  <a href="
     134             :  *  http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.699.1161&rep=rep1&type=pdf
     135             :  *  "> here </a>.
     136             :  *
     137             :  *  Hesthaven's formula is general in the degree of the shape functions used,
     138             :  *  so for our purposes we take the special case where the shape functions are
     139             :  *  linear in the interpolation variable, and define new variables accordingly.
     140             :  *  However, our interpolation variables do not necessarily have to be the
     141             :  *  logical coordinates themselves, though they often are. To make this
     142             :  *  distinction clear, we will define the new interpolation variables
     143             :  *  \f$\{\tilde{\xi},\tilde{\eta},\tilde{\zeta}\}\f$, which may either be the
     144             :  *  logical coordinates themselves or a invertible transformation of them. For
     145             :  *  the purposes of the bulged cube map, this transformation will be the same
     146             :  *  transformation that takes the logical coordinates into the equiangular
     147             :  *  coordinates. We will later see how this choice can lead to simplifications
     148             :  *  in the final map.
     149             :  *
     150             :  *  We define the following variables for
     151             :  *  \f$\alpha, \beta, \gamma \in\{\tilde{\xi},\tilde{\eta},\tilde{\zeta}\}\f$:
     152             :  *
     153             :  *  \f[
     154             :  *  f^{\pm}_{\alpha} = \frac{1}{2}(1\pm\alpha)\\
     155             :  *  f^{\pm\pm}_{\alpha \ \beta} = \frac{1}{4}(1\pm\alpha)(1\pm\beta)\\
     156             :  *  f^{\pm\pm\pm}_{\alpha \ \beta \ \gamma} =
     157             :  *  \frac{1}{8}(1\pm\alpha)(1\pm\beta)(1\pm\gamma)
     158             :  *  \f]
     159             :  *
     160             :  *  The formula involves six surfaces, which we will denote by
     161             :  *  \f$\vec{\sigma}\f$, twelve curves, denoted by \f$\vec{\Gamma}\f$, and eight
     162             :  *  vertices, denoted by \f$\vec{\pi}\f$, with the subscripts denoting which
     163             :  *  face(s) these objects belong to. The full volume map is given by:
     164             :  *
     165             :  *  \f{align*}
     166             :  *  \vec{x}(\xi,\eta,\zeta) = &
     167             :  *  f^{+}_{\tilde{\zeta}}\vec{\sigma}_{+\zeta}(\xi, \eta)+
     168             :  *  f^{-}_{\tilde{\zeta}}\vec{\sigma}_{-\zeta}(\xi, \eta)\\
     169             :  *  &+ f^{+}_{\tilde{\eta}}\vec{\sigma}_{+\eta}(\xi, \zeta)+
     170             :  *  f^{-}_{\tilde{\eta}}\vec{\sigma}_{-\eta}(\xi, \zeta)+
     171             :  *  f^{+}_{\tilde{\xi}}\vec{\sigma}_{+\xi}(\eta, \zeta)+
     172             :  *  f^{-}_{\tilde{\xi}}\vec{\sigma}_{-\xi}(\eta, \zeta)\\
     173             :  *  &- f^{++}_{\tilde{\xi} \ \tilde{\eta}}\vec{\Gamma}_{+\xi +\eta}(\zeta)-
     174             :  *  f^{-+}_{\tilde{\xi} \ \tilde{\eta}}\vec{\Gamma}_{-\xi +\eta}(\zeta)-
     175             :  *  f^{+-}_{\tilde{\xi} \ \tilde{\eta}}\vec{\Gamma}_{+\xi -\eta}(\zeta)-
     176             :  *  f^{--}_{\tilde{\xi} \ \tilde{\eta}}\vec{\Gamma}_{-\xi -\eta}(\zeta)\\
     177             :  *  &- f^{++}_{\tilde{\xi} \ \tilde{\zeta}}\vec{\Gamma}_{+\xi +\zeta}(\eta)-
     178             :  *  f^{-+}_{\tilde{\xi} \ \tilde{\zeta}}\vec{\Gamma}_{-\xi +\zeta}(\eta)-
     179             :  *  f^{+-}_{\tilde{\xi} \ \tilde{\zeta}}\vec{\Gamma}_{+\xi -\zeta}(\eta)-
     180             :  *  f^{--}_{\tilde{\xi} \ \tilde{\zeta}}\vec{\Gamma}_{-\xi -\zeta}(\eta)\\
     181             :  *  &- f^{++}_{\tilde{\eta} \ \tilde{\zeta}}\vec{\Gamma}_{+\eta +\zeta}(\xi)-
     182             :  *  f^{-+}_{\tilde{\eta} \ \tilde{\zeta}}\vec{\Gamma}_{-\eta +\zeta}(\xi)-
     183             :  *  f^{+-}_{\tilde{\eta} \ \tilde{\zeta}}\vec{\Gamma}_{+\eta -\zeta}(\xi)-
     184             :  *  f^{--}_{\tilde{\eta} \tilde{\zeta}}\vec{\Gamma}_{-\eta -\zeta}(\xi)\\
     185             :  *  &+ f^{+++}_{\tilde{\xi} \ \tilde{\eta} \ \tilde{\zeta}}\vec{\pi}_{+\xi +\eta
     186             :  * +\zeta}+ f^{-++}_{\tilde{\xi} \ \tilde{\eta} \ \tilde{\zeta}}\vec{\pi}_{-\xi
     187             :  * +\eta +\zeta}+ f^{+-+}_{\tilde{\xi} \ \tilde{\eta} \
     188             :  * \tilde{\zeta}}\vec{\pi}_{+\xi -\eta +\zeta}+
     189             :  *  f^{--+}_{\tilde{\xi} \ \tilde{\eta} \ \tilde{\zeta}}\vec{\pi}_{-\xi -\eta
     190             :  * +\zeta}\\
     191             :  *  &+ f^{++-}_{\tilde{\xi} \ \tilde{\eta} \ \tilde{\zeta}}\vec{\pi}_{+\xi +\eta
     192             :  * -\zeta}+ f^{-+-}_{\tilde{\xi} \ \tilde{\eta} \ \tilde{\zeta}}\vec{\pi}_{-\xi
     193             :  * +\eta -\zeta}+ f^{+--}_{\tilde{\xi} \ \tilde{\eta} \
     194             :  * \tilde{\zeta}}\vec{\pi}_{+\xi -\eta -\zeta}+ f^{---}_{\tilde{\xi} \
     195             :  * \tilde{\eta} \ \tilde{\zeta}}\vec{\pi}_{-\xi -\eta -\zeta} \f}
     196             :  *
     197             :  *
     198             :  *  ### The Special Case for Octahedral Symmetry
     199             :  *  The general formula is for the case in which there are six independently
     200             :  *  specified bounding surfaces. In our case, the surfaces are obtained by
     201             :  *  rotations and reflections of the upper-\f$\zeta\f$ face.
     202             :  *
     203             :  * We define the matrices corresponding to these transformations to be:
     204             :  *
     205             :  * \f[
     206             :  * S_{xy} =
     207             :  *  \begin{bmatrix}
     208             :  *  0 & 1 & 0\\
     209             :  *  1 & 0 & 0\\
     210             :  *  0 & 0 & 1\\
     211             :  *  \end{bmatrix},\
     212             :  *
     213             :  * S_{xz} =
     214             :  *  \begin{bmatrix}
     215             :  *  0 & 0 & 1\\
     216             :  *  0 & 1 & 0\\
     217             :  *  1 & 0 & 0\\
     218             :  *  \end{bmatrix},\
     219             :  *
     220             :  * S_{yz} =
     221             :  *  \begin{bmatrix}
     222             :  *  1 & 0 & 0\\
     223             :  *  0 & 0 & 1\\
     224             :  *  0 & 1 & 0\\
     225             :  *  \end{bmatrix}\f]
     226             :  *
     227             :  * \f[C_{zxy} =
     228             :  *  \begin{bmatrix}
     229             :  *  0 & 0 & 1\\
     230             :  *  1 & 0 & 0\\
     231             :  *  0 & 1 & 0\\
     232             :  *  \end{bmatrix},\
     233             :  *
     234             :  * C_{yzx} =
     235             :  *  \begin{bmatrix}
     236             :  *  0 & 1 & 0\\
     237             :  *  0 & 0 & 1\\
     238             :  *  1 & 0 & 0\\
     239             :  *  \end{bmatrix}\f]
     240             :  *
     241             :  * \f[N_{x} =
     242             :  *  \begin{bmatrix}
     243             :  *  -1 & 0 & 0\\
     244             :  *  0 & 1 & 0\\
     245             :  *  0 & 0 & 1\\
     246             :  *  \end{bmatrix},\
     247             :  *
     248             :  * N_{y} =
     249             :  *  \begin{bmatrix}
     250             :  *  1 & 0 & 0\\
     251             :  *  0 & -1 & 0\\
     252             :  *  0 & 0 & 1\\
     253             :  *  \end{bmatrix},\
     254             :  *
     255             :  * N_{z} =
     256             :  *  \begin{bmatrix}
     257             :  *  1 & 0 & 0\\
     258             :  *  0 & 1 & 0\\
     259             :  *  0 & 0 & -1\\
     260             :  *  \end{bmatrix}
     261             :  *  \f]
     262             :  *
     263             :  * The surface maps can now all be written in terms of
     264             :  * \f$\vec{\sigma}_{+\zeta}\f$ and these matrices:
     265             :  * <center>
     266             :  * \f$\vec{\sigma}_{-\zeta}(\xi, \eta) = N_z\vec{\sigma}_{+\zeta}(\xi, \eta)\\
     267             :  * \vec{\sigma}_{+\eta}(\xi, \zeta) = S_{yz}\vec{\sigma}_{+\zeta}(\xi, \zeta)\\
     268             :  * \vec{\sigma}_{-\eta}(\xi, \zeta) = N_yS_{yz}\vec{\sigma}_{+\zeta}(\xi,
     269             :  * \zeta)\\
     270             :  * \vec{\sigma}_{+\xi}(\eta, \zeta) = C_{zxy}\vec{\sigma}_{+\zeta}(\eta,
     271             :  * \zeta)\\
     272             :  * \vec{\sigma}_{-\xi}(\eta, \zeta) = N_xC_{zyx}\vec{\sigma}_{+\zeta}(\eta,
     273             :  * \zeta)\f$
     274             :  * </center>
     275             :  *
     276             :  * The four bounding curves \f$\vec{\Gamma}\f$ on the \f$+\zeta\f$ face are
     277             :  * given by:
     278             :  *
     279             :  * <center>
     280             :  * \f$\vec{\Gamma}_{+\xi,+\zeta}(\eta) = \vec{\sigma}_{+\zeta}(+1,\eta)\\
     281             :  * \vec{\Gamma}_{-\xi,+\zeta}(\eta) = \vec{\sigma}_{+\zeta}(-1,\eta)
     282             :  * = N_x\vec{\sigma}_{+\zeta}(+1, \eta)\\
     283             :  * \vec{\Gamma}_{+\eta,+\zeta}(\xi) = \vec{\sigma}_{+\zeta}(\xi,+1)
     284             :  * = S_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     285             :  * \vec{\Gamma}_{-\eta,+\zeta}(\xi) = \vec{\sigma}_{+\zeta}(\xi,-1)
     286             :  * = N_yS_{xy}\vec{\sigma}_{+\zeta}(+1,\xi)\f$
     287             :  * </center>
     288             :  *
     289             :  * The bounding curves on the other surfaces can be obtained by transformations
     290             :  * on the \f$+\zeta\f$ face:
     291             :  *
     292             :  * <center>
     293             :  * \f$\vec{\Gamma}_{+\xi,-\zeta}(\eta) = N_z\vec{\sigma}_{+\zeta}(+1,\eta)\\
     294             :  * \vec{\Gamma}_{-\xi,-\zeta}(\eta) = N_z\vec{\sigma}_{+\zeta}(-1,\eta)
     295             :  * = N_zN_x\vec{\sigma}_{+\zeta}(+1,\eta)\\
     296             :  * \vec{\Gamma}_{+\eta,-\zeta}(\xi) = N_z\vec{\sigma}_{+\zeta}(\xi,+1)
     297             :  * = N_zS_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     298             :  * \vec{\Gamma}_{-\eta,-\zeta}(\xi) = N_z\vec{\sigma}_{+\zeta}(\xi,-1)
     299             :  * = N_zN_yS_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     300             :  * \vec{\Gamma}_{+\xi,+\eta}(\zeta) =
     301             :  * C_{zxy}\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     302             :  * \vec{\Gamma}_{-\xi,+\eta}(\zeta) =
     303             :  * N_xC_{zxy}\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     304             :  * \vec{\Gamma}_{+\xi,-\eta}(\zeta) = C_{zxy}\vec{\sigma}_{+\zeta}(-1,\zeta)
     305             :  * = C_{zxy}N_x\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     306             :  * \vec{\Gamma}_{-\xi,-\eta}(\zeta) = N_xC_{zxy}\vec{\sigma}_{+\zeta}(-1,\zeta)
     307             :  * = N_xC_{zxy}N_x\vec{\sigma}_{+\zeta}(+1,\zeta)\f$
     308             :  * </center>
     309             :  *
     310             :  * Now we can write the volume map in terms of
     311             :  * \f$\vec{\sigma}_{+\zeta}\f$ only:
     312             :  * \f{align*}\vec{x}(\xi,\eta,\zeta) = &
     313             :  * (f^{+}_{\tilde{\zeta}} + f^{-}_{\tilde{\zeta}}N_z)
     314             :  * \vec{\sigma}_{+\zeta}(\xi, \eta)\\
     315             :  * &+ (f^{+}_{\tilde{\eta}} + f^{-}_{\tilde{\eta}}N_y)
     316             :  * S_{yz}\vec{\sigma}_{+\zeta}(\xi, \zeta)\\
     317             :  * &+ (f^{+}_{\tilde{\xi}} + f^{-}_{\tilde{\xi}}N_x)
     318             :  * C_{zxy}\vec{\sigma}_{+\zeta}(\eta, \zeta)\\
     319             :  * &- (f^{+}_{\tilde{\xi}}+f^{-}_{\tilde{\xi}}N_x)
     320             :  * (f^{+}_{\tilde{\eta}}+f^{-}_{\tilde{\eta}}N_y)
     321             :  * C_{zxy}\vec{\sigma}_{+\zeta}(+1, \zeta)\\
     322             :  * &- (f^{+}_{\tilde{\zeta}}+f^{-}_{\tilde{\zeta}}N_z)\left\{
     323             :  * (f^{+}_{\tilde{\xi}}+f^{-}_{\tilde{\xi}}N_x)\vec{\sigma}_{+\zeta}(+1, \eta)+
     324             :  * (f^{+}_{\tilde{\eta}}+f^{-}_{\tilde{\eta}}N_y)S_{xy}\vec{\sigma}_{+\zeta}(+1,
     325             :  * \xi)\right\}\\
     326             :  * &+ \frac{r}{\sqrt{3}}\vec{\tilde{\xi}}
     327             :  *  \f}
     328             :  *
     329             :  * Note that we can now absorb all of the \f$f\f$s into the matrix prefactors
     330             :  * in the above equation and obtain a final set of matrices. We define the
     331             :  * following *blending matrices*:
     332             :  *
     333             :  *  \f[
     334             :  *  B_{\tilde{\xi}} =
     335             :  *  \begin{bmatrix}
     336             :  *  0 & 0 & \tilde{\xi}\\
     337             :  *  1 & 0 & 0\\
     338             :  *  0 & 1 & 0\\
     339             :  *  \end{bmatrix},\
     340             :  *
     341             :  *  B_{\tilde{\eta}} =
     342             :  *  \begin{bmatrix}
     343             :  *  1 & 0 & 0\\
     344             :  *  0 & 0 & \tilde{\eta}\\
     345             :  *  0 & 1 & 0\\
     346             :  *  \end{bmatrix},\
     347             :  *
     348             :  *  B_{\tilde{\zeta}} =
     349             :  *  \begin{bmatrix}
     350             :  *  1 & 0 & 0\\
     351             :  *  0 & 1 & 0\\
     352             :  *  0 & 0 & \tilde{\zeta}\\
     353             :  *  \end{bmatrix}\\
     354             :  *
     355             :  *  B_{\tilde{\xi}\tilde{\eta}} =
     356             :  *  \begin{bmatrix}
     357             :  *  0 & 0 & \tilde{\xi}\\
     358             :  *  \tilde{\eta} & 0 & 0\\
     359             :  *  0 & 1 & 0\\
     360             :  *  \end{bmatrix},\
     361             :  *
     362             :  *  B_{\tilde{\xi}\tilde{\zeta}} =
     363             :  *  \begin{bmatrix}
     364             :  *  \tilde{\xi} & 0 & 0\\
     365             :  *  0 & 1 & 0\\
     366             :  *  0 & 0 & \tilde{\zeta}\\
     367             :  *  \end{bmatrix},\
     368             :  *
     369             :  *  B_{\tilde{\eta}\tilde{\zeta}} =
     370             :  *  \begin{bmatrix}
     371             :  *  0 & 1 & 0\\
     372             :  *  \tilde{\eta} & 0 & 0\\
     373             :  *  0 & 0 & \tilde{\zeta}\\
     374             :  *  \end{bmatrix}\\
     375             :  *
     376             :  *  B_{\tilde{\xi}\tilde{\eta}\tilde{\zeta}} =
     377             :  *  \begin{bmatrix}
     378             :  *  \tilde{\xi} & 0 & 0\\
     379             :  *  0 & \tilde{\eta} & 0\\
     380             :  *  0 & 0 & \tilde{\zeta}\\
     381             :  *  \end{bmatrix}
     382             :  *  \f]
     383             :  *
     384             :  *  Now we can write the volume map in these terms:
     385             :  *
     386             :  * \f{align*}
     387             :  * \vec{x}(\xi,\eta,\zeta) = &
     388             :  * B_{\tilde{\zeta}}
     389             :  * \vec{\sigma}_{+\zeta}(\xi, \eta)\\& +
     390             :  * B_{\tilde{\eta}}
     391             :  * \vec{\sigma}_{+\zeta}(\xi, \zeta)+
     392             :  * B_{\tilde{\xi}}
     393             :  * \vec{\sigma}_{+\zeta}(\eta, \zeta)\\& -
     394             :  * B_{\tilde{\xi} \tilde{\eta}}
     395             :  * \vec{\sigma}_{+\zeta}(+1, \zeta)-
     396             :  * B_{\tilde{\xi} \tilde{\zeta}}
     397             :  * \vec{\sigma}_{+\zeta}(+1, \eta)+
     398             :  * B_{\tilde{\eta} \tilde{\zeta}}
     399             :  * \vec{\sigma}_{+\zeta}(+1, \xi)\\& +
     400             :  * B_{\tilde{\xi} \tilde{\eta} \tilde{\zeta}}
     401             :  * \vec{\sigma}_{+\zeta}(+1, +1)
     402             :  * \f}
     403             :  *
     404             :  * ### The Bulged Cube Map
     405             :  * We now use the result above to provide the mapping for the bulged cube.
     406             :  * First we will define the variables \f$\rho_A\f$ and \f$\rho_{AB}\f$, for
     407             :  * \f$A, B \in \{\Xi,\mathrm{H}, \mathrm{Z}\} \f$, where \f$\mathrm{Z}\f$
     408             :  * is \f$\tan(\zeta\pi/4)\f$ in the equiangular case and \f$\zeta\f$ in the
     409             :  * equidistant case:
     410             :  *
     411             :  * \f[
     412             :  * \rho_A = \sqrt{2 + A^2}\\
     413             :  * \rho_{AB} = \sqrt{1 + A^2 + B^2}
     414             :  * \f]
     415             :  * The final mapping is then:
     416             :  * \f[
     417             :  * \vec{x}(\xi,\eta,\zeta) = \frac{(1-s)R}{\sqrt{3}}
     418             :  * \begin{bmatrix}
     419             :  * \Xi\\
     420             :  * \mathrm{H}\\
     421             :  * \mathrm{Z}\\
     422             :  * \end{bmatrix} +
     423             :  * \frac{sR}{\sqrt{3}}
     424             :  * \begin{bmatrix}
