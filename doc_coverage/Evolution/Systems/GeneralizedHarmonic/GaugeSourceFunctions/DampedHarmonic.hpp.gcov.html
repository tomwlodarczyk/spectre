      49             :  * \details Computes the gauge source function designed for binary black hole

      53             :  * The covariant form of the source function \f$H_a\f$ is written as:

      64             :  * 3-slice, i.e. \f$g_{ab} = \psi_{ab} + t_a t_b\f$). The prefactors are:
      65             :  *
      66             :  * \f{align*}
      67             :  *  \mu_{L1} &= A_{L1} R_{L1}(t) W(x^i) \mathrm{log}(\sqrt{g}/N)^{e_{L1}}, \\
      68             :  *  \mu_{L2} &= A_{L2} R_{L2}(t) W(x^i) \mathrm{log}(1/N)^{e_{L2}}, \\
      69             :  *  \mu_{S} &= A_{S} R_{S}(t) W(x^i) \mathrm{log}(\sqrt{g}/N)^{e_{S}},
      70             :  * \f}
      71             :  *
      72             :  * temporal roll-on functions \f$ R_X(t)\f$ (with label \f$ X\f$) are:
      73             :  *
      74             :  * \f{align*}

      76             :  *     R_X(t) & = 1 - \exp[-((t - t_{0,X})/ \sigma_t^X)^4], & t\geq t_{0,X} \\
