     148             :  * \brief Computes the generalized-harmonic upwind flux
     149             :  *
     150             :  * The upwind flux in general is given by Eq. (6.3) of \cite Teukolsky2015ega :
     151             :  * \f{eqnarray}{
     152             :  * D = S \Lambda^{\rm pos} S^{-1} u^{\rm int}
     153             :  *   + S \Lambda^{\rm neg} S^{-1} u^{\rm ext},
     154             :  * \f}
     155             :  * where \f$u\f$ is a vector of the evolved variables, \f$S^{-1}\f$
     156             :  * maps the evolved variables into the characteristic variables \f$S^{-1}u\f$,
     157             :  * \f$S\f$ maps the characteristic variables into the evolved variables,
     158             :  * and \f$\Lambda\f$ is a diagonal matrix of the average characteristic
     159             :  * speed at the interface.
     160             :  *
     161             :  * Here, \f$S^{-1}u^{\rm int}\f$ represents the
     162             :  * characteristic variables at the element interface, computed using
     163             :  * evolved variables from the element interior;
     164             :  * \f$S^{-1}u^{\rm ext}\f$ represents the characteristic variables at the
     165             :  * element interface, computed using evolved variables from the exterior,
     166             :  * neighboring element; \f$\Lambda^{\rm pos}\f$ is a diagonal
     167             :  * matrix whose nonzero entries are the average characteristic speeds
     168             :  * that are positive ("outgoing", i.e. leaving the element); and
     169             :  * \f$\Lambda^{\rm neg}\f$ is a diagonal matrix whose nonzero entries are
     170             :  * the average characteristic speeds that are negative ("incoming", i.e.
     171             :  * entering the element). If a characteristic field
     172             :  * \f$U^{\rm int}\f$ has a characteristic speed \f$v^{\rm int}\f$ and the same
     173             :  * field in the exterior \f$U^{\rm ext}\f$ has speed \f$v^{\rm ext}\f$, then
     174             :  * the average characteristic speed is
     175             :  * \f$v^{\rm avg} = (1/2)(v^{\rm int} + v^{\rm ext})\f$.
     176             :  *
     177             :  * This function implements the upwind flux for the generalized harmonic
     178             :  * system. First, it computes the characteristic variables using i) the
     179             :  * evolved variables from the interior and ii) the evolved variables from the
     180             :  * exterior. Then, it computes \f$\Lambda^{\rm pos}\f$ and
     181             :  * \f$\Lambda^{\rm neg}\f$ from the average characteristic speeds.
     182             :  * Then, it computes the combination
     183             :  * \f$\Lambda^{\rm pos} S^{-1} u^{\rm int}
     184             :  * + \Lambda^{\rm neg} S^{-1} u^{\rm ext}\f$. Finally, it
     185             :  * applies \f$S\f$ by converting the result back from characteristic to
     186             :  * evolved variables, using the unit normal vector of the element and
     187             :  * the average value of the field
     188             :  * \f$\gamma_2^{\rm avg} = (1/2)(\gamma_2^{\rm int} + \gamma_2^{\rm ext})\f$,
     189             :  * where here \f$\gamma_2^{\rm int}\f$ is the value of \f$\gamma_2\f$
     190             :  * in the interior and \f$\gamma_2^{\rm ext}\f$ is the value of
     191             :  * \f$\gamma_2\f$ in the exterior.
     192             :  *
     193             :  * \warning as currently implemented, this flux does not satisfy the
     194             :  * generalized Rankine-Hugoniot condition
     195             :  * \f$D^{\rm int}([S^{-1} u]^{\rm int}, [S^{-1} u]^{\rm ext})
     196             :  * = - D^{\rm ext}([S^{-1} u]^{\rm ext}, [S^{-1} u]^{\rm int})\f$,
     197             :  * which enforces that the flux leaving the interior is the flux entering
     198             :  * the exterior and vice versa. This condition is necessary for a
     199             :  * well-balanced scheme. Use this flux with caution; so far it has
     200             :  * been tested for evolving Kerr-Schild and a perturbed Kerr black hole.
