Detailed Description

template<size_t Dim>
struct Poisson::FirstOrderInternalPenaltyFlux< Dim >

The internal penalty flux for the first oder formulation of the Poisson equation.

Details

For the sourced equation this is \(-\frac{\nabla u_\mathrm{int} + \nabla u_\mathrm{ext}}{2} + \sigma \left(u_\mathrm{int} - u_\mathrm{ext}\right)\) and for the auxiliary equation it is \(\frac{ u_\mathrm{int} + u_\mathrm{ext}}{2}\). The penalty factor \(\sigma\) is responsible for removing zero eigenmodes and impacts the conditioning of the linear operator to be solved. It can be chosen as \(\sigma=C\frac{N_\mathrm{points}^2}{h}\) where \(N_\mathrm{points}\) is the number of collocation points (i.e. the polynomial degree plus 1), \(h\) is a measure of the element size in inertial coordinates and \(C\leq 1\) is a free parameter (see e.g. [23], section 7.2).
