95  "Creates a 3D spherical shell with 6 Blocks. `UseEquiangularMap` has\n"
96  "a default value of `true` because there is no central Block in this\n"
97  "domain. Equidistant coordinates are best suited to Blocks with\n"
98  "Cartesian grids. However, the option is allowed for testing "
99  "purposes. The `aspect_ratio` moves grid points on the shell towards\n"
100  "the equator for values greater than 1.0, and towards the poles for\n"
101  "positive values less than 1.0. If `UseLogarithmicMap` is set to true,\n"
102  "the radial gridpoints will be spaced uniformly in \f$log(r)\f$. The\n"
103  "user may also choose to use only a single wedge (along the -x\n"
104  "direction), or four wedges along the x-y plane using the `WhichWedges`\n"
105  "option. Using the RadialBlockLayers option, a user may increase the\n"
106  "number of Blocks in the radial direction."};
