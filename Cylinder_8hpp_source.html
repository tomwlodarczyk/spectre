22 /// Create a 3D Domain in the shape of a cylinder where the cross-section
23 /// is a square surrounded by four two-dimensional wedges (see Wedge2D).

82  static constexpr OptionString help{
83  "Creates a 3D Cylinder with five Blocks.\n"
84  "Only one refinement level for all dimensions is currently supported.\n"
85  "The number of gridpoints in each dimension can be set independently.\n"
86  "The number of gridpoints along the dimensions of the square is equal\n"
87  "to the number of gridpoints along the angular dimension of the wedges.\n"
88  "Equiangular coordinates give better gridpoint spacings in the angular\n"
89  "direction, while equidistant coordinates give better gridpoint\n"
90  "spacings in the center block. This Domain uses equidistant coordinates\n"
91  "by default. The boundary conditions are set to be periodic along the\n"
92  "cylindrical z-axis by default."};
