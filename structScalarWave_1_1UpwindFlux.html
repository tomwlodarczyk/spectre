Detailed Description

template<size_t Dim>
struct ScalarWave::UpwindFlux< Dim >

Computes the upwind flux.

The upwind flux is given by:

\begin{align} F^*(\Psi) =& 0 \\ F^*(\Pi) =& \frac{1}{2}\left(F(\Pi)_{\mathrm{int}} + F(\Pi)_{\mathrm{ext}} + \Pi_{\mathrm{int}} - \Pi_{\mathrm{ext}}\right) \\ F^*(\Phi_i) =& \frac{1}{2} \left(F(\Phi_i)_{\mathrm{int}} + F(\Phi_i)_{\mathrm{ext}} + (n_i)_{\mathrm{int}} F(\Pi)_{\mathrm{int}} - (n_i)_{\mathrm{ext}} F(\Pi)_{\mathrm{ext}}\right) \end{align}

where \(F^*\) is the normal dotted with the numerical flux and \(F\) is the normal dotted with the flux, which is computed in ScalarWave::ComputeNormalDotFluxes
