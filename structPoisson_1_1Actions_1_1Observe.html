Detailed Description

Action to observe volume and reduction data for the Poisson system.

This action observes the following:

  • Reduction data:
    • "Iteration": The linear solver iteration
    • "NumberOfPoints": The total number of grid points across all elements
    • "L2Error": The standard L2 vector norm of the pointwise difference between the numerical and analytic solution \(\sqrt{\sum_i \left( u^\mathrm{numerical}_i - u^\mathrm{analytic}_i\right)^2}\) over the grid points across all elements.
  • Volume data:
    • Poisson::Tags::Field::name(): The numerical solution \(u^\mathrm{numerical}\)
    • Poisson::Tags::Field::name() + "Analytic": The analytic solution \(u^\mathrm{analytic}\)
    • Poisson::Tags::Field::name() + "Error": The pointwise error \(u^\mathrm{numerical} - u^\mathrm{analytic}\)
    • "InertialCoordinates_{x,y,z}"

Uses:

  • Metavariables:
    • analytic_solution_tag
    • Items required by observers::Observer<Metavariables>
  • DataBox:
    • LinearSolver::Tags::IterationId
    • Tags::Mesh<Dim>
    • Poisson::Tags::Field
    • Tags::Coordinates<Dim, Frame::Inertial>
Note
This action can be adjusted before compiling an executable to observe only the desired quantities.
