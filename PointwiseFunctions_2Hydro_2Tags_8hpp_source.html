195 /// The vector \f$J^i\f$ in \f$\dot{M} = -\int J^i s_i d^2S\f$,
196 /// representing the mass flux through a surface with normal \f$s_i\f$.
197 ///
198 /// Note that the integral is understood
199 /// as a flat-space integral: all metric factors are included in \f$J^i\f$.
200 /// In particular, if the integral is done over a Strahlkorper, the
201 /// `StrahlkorperGr::euclidean_area_element` of the Strahlkorper should be used,
202 /// and \f$s_i\f$ is
203 /// the normal one-form to the Strahlkorper normalized with the flat metric,
204 /// \f$s_is_j\delta^{ij}=1\f$.
205 ///
206 /// The formula is
207 /// \f$ J^i = \rho W \sqrt{\gamma}(\alpha v^i-\beta^i)\f$,
208 /// where \f$\rho\f$ is the mass density, \f$W\f$ is the Lorentz factor,
209 /// \f$v^i\f$ is the spatial velocity of the fluid,
210 /// \f$\gamma\f$ is the determinant of the 3-metric \f$\gamma_{ij}\f$,
211 /// \f$\alpha\f$ is the lapse, and \f$\beta^i\f$ is the shift.
212 template <typename DataType, size_t Dim, typename Fr>
213 struct MassFlux : db::SimpleTag {
214  using type = tnsr::I<DataType, Dim, Fr>;
215  static std::string name() noexcept {
216  return Frame::prefix<Fr>() + "MassFlux";
217  }
218 };
