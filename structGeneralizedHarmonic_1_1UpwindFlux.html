Detailed Description

template<size_t Dim>
struct GeneralizedHarmonic::UpwindFlux< Dim >

Computes the generalized-harmonic upwind flux.

The upwind flux in general is given by Eq. (6.3) of [43] :

\begin{eqnarray} D = S \Lambda^{\rm pos} S^{-1} u^{\rm int} + S \Lambda^{\rm neg} S^{-1} u^{\rm ext}, \end{eqnarray}

where \(u\) is a vector of the evolved variables, \(S^{-1}\) maps the evolved variables into the characteristic variables \(S^{-1}u\), \(S\) maps the characteristic variables into the evolved variables, and \(\Lambda\) is a diagonal matrix of the average characteristic speed at the interface.

Here, \(S^{-1}u^{\rm int}\) represents the characteristic variables at the element interface, computed using evolved variables from the element interior; \(S^{-1}u^{\rm ext}\) represents the characteristic variables at the element interface, computed using evolved variables from the exterior, neighboring element; \(\Lambda^{\rm pos}\) is a diagonal matrix whose nonzero entries are the average characteristic speeds that are positive ("outgoing", i.e. leaving the element); and \(\Lambda^{\rm neg}\) is a diagonal matrix whose nonzero entries are the average characteristic speeds that are negative ("incoming", i.e. entering the element). If a characteristic field \(U^{\rm int}\) has a characteristic speed \(v^{\rm int}\) and the same field in the exterior \(U^{\rm ext}\) has speed \(v^{\rm ext}\), then the average characteristic speed is \(v^{\rm avg} = (1/2)(v^{\rm int} + v^{\rm ext})\).

This function implements the upwind flux for the generalized harmonic system. First, it computes the characteristic variables using i) the evolved variables from the interior and ii) the evolved variables from the exterior. Then, it computes \(\Lambda^{\rm pos}\) and \(\Lambda^{\rm neg}\) from the average characteristic speeds. Then, it computes the combination \(\Lambda^{\rm pos} S^{-1} u^{\rm int} + \Lambda^{\rm neg} S^{-1} u^{\rm ext}\). Finally, it applies \(S\) by converting the result back from characteristic to evolved variables, using the unit normal vector of the element and the average value of the field \(\gamma_2^{\rm avg} = (1/2)(\gamma_2^{\rm int} + \gamma_2^{\rm ext})\), where here \(\gamma_2^{\rm int}\) is the value of \(\gamma_2\) in the interior and \(\gamma_2^{\rm ext}\) is the value of \(\gamma_2\) in the exterior.

Warning
as currently implemented, this flux does not satisfy the generalized Rankine-Hugoniot condition \(D^{\rm int}([S^{-1} u]^{\rm int}, [S^{-1} u]^{\rm ext}) = - D^{\rm ext}([S^{-1} u]^{\rm ext}, [S^{-1} u]^{\rm int})\), which enforces that the flux leaving the interior is the flux entering the exterior and vice versa. This condition is necessary for a well-balanced scheme. Use this flux with caution; so far it has been tested for evolving Kerr-Schild and a perturbed Kerr black hole.
